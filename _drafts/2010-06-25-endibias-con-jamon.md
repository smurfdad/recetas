Para 2 personas.

<h3>Ingredientes</h3>

4 ENDIBIAS

JAMON SERRANO CORTADO EN DADOS

MANTEQUILLA

SAL

1 CHORRITO DE BRANDY

QUESO RALLADO

<h3>Preparación</h3>

SE LAVAN LAS ENDIBIAS. EN UNA CAZUELA SE FUNDE LA MANTEQUILLA Y SE AñADEN LAS ENDIBIAS HúMEDAS DE AGUA. SE AñADE EL JAMóN TROCEADO Y UN POCO DE SAL. SE DEJA QUE HIERVA A FUEGO LENTO. DESPUéS SE LE AñADE UN CHORRITO DE BRANDY, Y UN POCO DE AGUA PARA QUE SE TERMINEN DE HACER. CUANDO YA ESTáN HECHAS SE LES PONE EL QUESO RALLADO POR ENCIMA Y UNA VEZ FUNDIDO ESTAN LISTAS PARA SERVIR.
